$ npm i -g git-commit-auto
user@dev-machine ~

$ git commit-auto

Engineered for flow state.

Minimal configuration. Maximum output.

Zero Latency

Powered by Gemini Flash. Generates context-aware messages faster than you can type "fix typo".

Conventional Spec

Strict adherence to Conventional Commits. Your changelogs will finally look professional.

One Flag Push

Run git commit-auto push to stage, commit, and push in a single keystroke.

Get Started

Requires a free Google Gemini API key.

1

Get your API Key

Visit Google AI Studio to grab a free key.

aistudio.google.com →
2

Export Variable

Add this to your .zshrc or .bashrc

export GEMINI_API_KEY="your-key-here"
3

Install Script

Download and move to your bin folder.

curl -L https://raw.githubusercontent.com/user/repo/main/install.sh | bash
